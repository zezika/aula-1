HTMLLinguagem de Marcação de hipertexto

Tutorial para iniciantes

HTML (liguaguem de marcação de hipertexto) é o bloco de construção mais básico da web. Define o
significado e a estrutura do coteúdo de web. Outras tecnologias além do HTML geralmente são
usadas para descrever a aparencia/apresentação (CSS) ou a funcionalidade/comportamento
(JavaScript) de uma pagina web
